# 対多 (不和)

唯一偽中国語使用可能掲示板

🌱 草。非常愉快、爆笑投稿用
👍 親指。良仕事、高評価投稿用
🍬 飴。可哀想、同情、及元気譲渡用

利用可能、不和鯖内！

## 起動方法

設置不和機器鍵、主番号:

```bash
cp .env.example .env
```

生成、起動:

```bash
npm run build
npm run start
```
